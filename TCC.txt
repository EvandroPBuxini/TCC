Ideia do Projeto: EstéticaBot – Assistente Inteligente para Clínicas Estéticas

Descrição:

O EstéticaBot é um chatbot baseado em IA capaz de atender clientes de uma clínica estética, fornecendo informações sobre serviços, horários disponíveis, preços e políticas da clínica. Além disso, ele permitirá o agendamento automático de consultas e enviará notificações aos clientes para confirmação e lembrete dos atendimentos. O sistema utilizará diversas tecnologias da AWS para garantir escalabilidade, segurança e eficiência.

Stakeholders:

•	Mulheres
•	Esteticistas
•	Manicures

Justificativa:

A implementação de um sistema que permita aos clientes agendar e tirar dúvidas referente a serviços estéticos. Por meio deste sistema, o cliente poderá entender melhor os procedimentos, por relaizar perguntas e saber para que serve cada procedimento. Poderá também realizar agendamento dos serviços.

2.	Levantamento de Requisitos

Requisitos Funcionais:

•	O sistema deve permitir que o cliente realize agendamento de serviços.
•	O sistema deve fornecer informações sobre procedimentos por meio de perguntas realizadas.
•	O sistema deve alertar se caso o horário desejado não estiver disponível.

Requisitos Não Funcionais:

•	O sistema deve ser acessível via dispositivos móveis e desktop.
•	O sistema deve garantir a privacidade e segurança dos dados dos usuários.

MVP (Produto Mínimo Viável):

•	Agendamento de serviços e informações sobre cada serviço.
 
3.	Planeiamento Ágil

Backlog de Tarefas:

1.	Pesquisa sobre horários disponíveis.
2.	Informações sobre os tipos de serviços prestados.
3.	Serviço de notificação de agendamento.
